# Media library tracker
a lightweight web application to manage and track progress for Books and TV Shows using a unified SQLite database

# Feature
- Unified Database: One central hub for all media types using SQL table inheritance.

- Progress Tracking: Track pages read for books and episodes watched for shows.

- Dynamic UI: Responsive HTML info cards that change color/content based on media type.

  Database: SQLite3

  Frontend: HTML5, CSS3 (Flexbox/Grid)
